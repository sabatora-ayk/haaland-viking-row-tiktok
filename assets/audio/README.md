# audio アセットについて(TikTok版)

著作権のある音源は組み込んでいません。以下のパスに、
著作権フリー(もしくは自作)の音源を配置してください。
配置しなくても動画は再生されます(AudioManagerがエラーを握りつぶします)。

- bgm-main.mp3      : メインBGM(開始ゲートのタップ直後から再生)
- se-unlock.mp3      : ステージ解放時のSE
- se-ballon-dor.mp3  : フィナーレ演出のSE
- se-row.mp3         : タップ(漕ぎ)のSE
- se-thunder.mp3     : 雷鳴のSE

差し替えは config/game-config.js の audio セクションのみで完結します。
